 ECOMMERCE SYSTEM (Procedural PHP + MySQL)

1) Database
   - Open phpMyAdmin
   - Import: database.sql
   - DB name: ecommerce_demo

2) Config
   - Open config.php and set your DB username/password if needed.

3) Run
   - Place folder "ecommerce" inside: C:\xampp\htdocs\
   - Open: http://localhost/ecommerce/

4) Admin
   - Register FIRST user (this becomes ADMIN automatically)
   - Admin login: http://localhost/ecommerce/admin/login.php
   - Add Category first, then add Products.

Notes:
- No default users, no default products.
- If there are no products, Home shows "Not yet" and Shop shows "No product yet".
